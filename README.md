## Usage
BukkitBootstrap can be run by executing: `java -jar BukkitBootstrap-{version}.jar --serverJar {server.jar}`

Example: `java -jar BukkitBootstrap-1.0.0.jar --serverJar spigot-1.12.1.jar`
